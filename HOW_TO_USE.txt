🎯 دا څه شی دی؟
----------------
یو آفلاین اپلیکیشن دی چې په Android موبایل کې د اپلیکیشن په څېر
کار کوي. تاسو د اسعارو نرخونه پخپله داخلوئ او محاسبه کوئ.

📦 څه شته پکې؟
----------------
• 12 افغانۍ، ډالر، یورو، پاونډ، درهم، تومان، روپۍ، یوان، ریال، لیره، روبل
• د خرید او فروش نرخ جلا جلا
• چټک محاسبه: مقدار + کرنسی ولیکئ → ټولې کرنسۍ راته ښکاره شي
• نوې کرنسی اضافه کول
• تاریخ (History)
• پښتو/انګلیسي دواړه ژبې
• 100% آفلاین — هېڅکله اینټرنېټ نه غواړي

📱 په Android کې نصبولو لارښوونه:
----------------------------------
لاره 1 — ساده (سپارښتنه):
  1. ټول فایلونه (index.html, style.css, app.js, sw.js,
     manifest.webmanifest) یوځای یو فولډر کې وساتئ
  2. index.html په Chrome یا Firefox کې خلاص کړئ
  3. په مینو کې "Add to Home Screen" یا "په کور پاڼه کې اضافه کړئ"
  4. اوس د اپلیکیشن په څېر کار کوي!

لاره 2 — له کمپیوټر څخه:
  1. فایلونه موبایل ته ولېږئ (WhatsApp، Email، USB)
  2. index.html په File Manager کې خلاص کړئ
  3. د لاره 1 مرحله 3 تعقیب کړئ

🚀 څنګه کار کوي؟
------------------
• لومړی ځل: ۱۲ کرنسۍ اتومات اضافه شوي (افغانۍ، ډالر، یورو...)
• 💾 خوندي تڼۍ: کله چې نرخونه بدل کړئ، خوندي یې کړئ
• ⚡ چټک محاسبه: مقدار ولیکئ (مثال: 300) + کرنسی وټاکئ (USD)
  → ټولې کرنسۍ راته ښکاره کېږي
• + کرنسی: نوې کرنسی اضافه کړئ
• 🗑️ تاریخ پاکول: زاړه تاریخ حذف کړئ

🔒 خونديتوب:
-----------
• ټول معلومات په موبایل کې ذخیره کېږي (هېڅ سرور ته نه ځي)
• اینټرنېټ ته اړتیا نشته — 100% آفلاین
• د خرید/فروش نرخ جلا مدیریت کوئ

د کمپیوټر لپاره:
------------------
یوازې index.html په براوزر کې خلاص کړئ (Chrome/Firefox/Edge) — کار کوي!

❓ کومه ستونزه وي؟
-------------------
• که اپلیکیشن نه خلاصېده: Service Worker د HTTPS غواړي
  → په GitHub Pages، Netlify، یا Vercel کې Deploy کړئ
  → یا په localhost کې چلوئ
• د Android کې د offline کار لپاره: لومړی ځل آنلاین خلاص کړئ
  بیا آفلاین هم کار کوي
